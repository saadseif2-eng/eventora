# Eventora

> AI-powered Event Experience Platform

Eventora is a SaaS platform that enables users to create beautiful interactive event websites without coding.

Unlike traditional invitation builders, Eventora uses a modular rendering engine powered by Themes, Layouts and Components.

---

# Vision

Build the easiest platform for creating premium event websites in minutes.

Users should only provide event information.

Eventora handles:

- Design
- Theme
- Layout
- Rendering
- Publishing
- RSVP
- Analytics
